# Privacy Policy for Markd

**Effective Date:** [3/13/2026]

## Data Collection

Markd does NOT collect, store, transmit, or share any personal data or user information.

## Local Storage

The extension uses browser local storage solely to save custom timestamps that users manually enter. This data:
- Remains entirely on your device
- Is never transmitted to external servers
- Is never shared with third parties
- Can be deleted at any time by clearing your browser data

## External API Calls

Markd makes anonymous API requests to SponsorBlock (sponsor.ajay.app) to fetch community-submitted chapter markers. These requests:
- Only send publicly available video IDs
- Do not include any personal information
- Are the same requests any user could make directly to SponsorBlock's public API

## Changes to Privacy Policy

Any changes to this privacy policy will be posted here and in extension updates.
